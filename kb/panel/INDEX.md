# Kirby Panel extensions

This KB covers Kirby Panel extension development (PHP + Vue) with a K5 -> K6 migration focus.

Version tags used in docs:

- `K5:` Kirby 5 behavior or tooling
- `K6:` Kirby 6 behavior or tooling
- `K5 -> K6:` migration guidance
